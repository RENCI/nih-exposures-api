## Sample data

Sample data is extracted from the initial CMAQ database prior to having the statistical information added.

```
COPY (SELECT * FROM cmaq_exposures_data
      WHERE utc_date_time >= '2011-01-01 00:00:00'
            AND utc_date_time < '2011-05-01 00:00:00'
            AND row >= '1'
            AND row < '5'
            AND col >= '1'
            AND col < '5')
TO '/var/lib/pgsql/cmaq-2010-2011.csv'
WITH CSV HEADER;
```
